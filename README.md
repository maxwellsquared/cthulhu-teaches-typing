## Good to knows

Start Rails server:
```
rails server
```

Reset the db:
```
/cd into backend/typing
rails db:drop
rails db:setup
```

In terminal
```bash
curl POST http://localhost:3000/login -H 'Content-Type: application/json' -d '{"password":"test","email":"adamgrharvey@gmail.com"}' -v
```
